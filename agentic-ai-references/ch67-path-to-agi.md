# Chapter 67: The Path to AGI — Agents as a Stepping Stone

## Core Thesis
Agency is a necessary condition for AGI. The spectrum: tool → assistant → agent → autonomous system → ASI. Open-ended learning and world models are the missing pieces.

## Key References

### AGI Definitions
- **OpenAI** — "Highly autonomous systems that outperform humans at most economically valuable work."
- **DeepMind** — "Theoretically, an AGI would be able to learn any intellectual task that a human being can."
- **Bostrom (2014)** — "Superintelligence".

### Current Frontier Capabilities
- **o3/o4-mini** — 600+ tool calls, multimodal reasoning, deliberative alignment.
- **DeepSeek-R1** — Pure RL emergence of reflection and verification.
- **InternAgent-1.5** — Autonomous scientific discovery.
- **Devin, OpenHands** — Autonomous software engineering.

### Open Problems
- **Long-horizon reliability** — Million-step tasks without errors.
- **World models** — Understanding physics, causality, social dynamics.
- **Value learning** — Inferring human preferences from observation.
- **Scalable oversight** — Supervising agents that exceed human capability.
- **Missing pieces** — What capabilities do agents still lack?
