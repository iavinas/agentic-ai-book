# Chapter 55: Data Analysis and Business Intelligence Agents

## Core Thesis
SQL generation, pandas code execution, chart generation, and statistical reasoning. The data analyst agent bridges natural language and data.

## Key References

### Text-to-SQL
- **Spider** — Complex and cross-domain semantic parsing. https://yale-lily.github.io/spider
- **BIRD** — Big bench for large-scale databases. https://bird-bench.github.io/
- **SQL-PaLM** — Google. https://arxiv.org/abs/2302.05930

### Data Analysis
- **Code interpreter pattern** — LLM writes pandas/NumPy code, executes, observes.
- **Visualization** — Generating charts and dashboards.
- **Statistical reasoning** — Interpreting results correctly.

### Agent Architectures
- **Schema understanding** — Inferring database structure from DDL.
- **Query planning** — Breaking complex questions into SQL joins and aggregations.
- **Result interpretation** — Explaining query results in natural language.
- **Iterative refinement** — Adjusting queries based on preliminary results.
