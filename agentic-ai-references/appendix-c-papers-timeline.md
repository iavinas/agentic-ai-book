# Appendix C: Key Papers Timeline

## 2022 — The Foundation
- **ReAct** — Yao et al. Reasoning + acting loop. https://arxiv.org/abs/2210.03629
- **Toolformer** — Schick et al. Teaching LLMs to use tools. https://arxiv.org/abs/2302.04761
- **Chain-of-Thought** — Wei et al. Step-by-step reasoning. https://arxiv.org/abs/2201.11903
- **InstructGPT / RLHF** — Ouyang et al. Alignment via human feedback. https://arxiv.org/abs/2203.02155
- **ChatGPT** — OpenAI. Conversational AI milestone.

## 2023 — The Agent Explosion
- **AutoGPT** — Significant Gravitas. First open-ended autonomous agent.
- **BabyAGI** — Nakajima. Task-driven autonomy.
- **Voyager** — Wang et al. Open-ended embodied agent. https://arxiv.org/abs/2305.16291
- **GPT-4** — OpenAI. Multimodal frontier model.
- **LLaMA** — Meta. Open foundation model.
- **DSPy** — Stanford. Declarative LLM programming. https://arxiv.org/abs/2310.03714
- **ToolLLM / Gorilla** — Massive API tool learning.
- **MetaGPT / ChatDev** — Multi-agent software teams.

## 2024 — Reasoning Emerges
- **o1** — OpenAI. First reasoning model trained with RL. https://openai.com/index/learning-to-reason-with-llms/
- **DPO** — Direct Preference Optimization. https://arxiv.org/abs/2305.18290
- **CrewAI** — Role-based agent orchestration.
- **Devin** — Cognition. AI software engineer.
- **SWE-bench** — Real GitHub issue resolution benchmark. https://arxiv.org/abs/2310.06770
- **Claude Computer Use** — Anthropic. Agentic computer control.
- **MCP** — Model Context Protocol. https://modelcontextprotocol.io/

## 2025 — The Agentic Inflection
- **DeepSeek-R1** — Pure RL reasoning emergence. Open weights. https://arxiv.org/abs/2501.12948
- **o3 / o4-mini** — Full tool use + reasoning. 600+ tool calls.
- **QwQ-32B** — Compact open-weight reasoning. https://huggingface.co/Qwen/QwQ-32B
- **Claude Computer Use GA** — Production-ready agentic control.
- **A2A** — Agent-to-Agent Protocol. Google / Linux Foundation.
- **OWASP Agentic Top 10** — Security framework for agents.
- **GAIA2** — Updated general AI assistant benchmark.
- **AgentPRM, PRIME, WebAgent-R1** — Process reward models and RL for agents.
- **OpenAI Deep Research** — Iterative web research agent.

## 2026 — Standards and Scale
- **AG2 Beta** — Successor to AutoGen. https://ag2.ai/
- **Microsoft Agent Framework** — Magentic, Agent Mesh.
- **A2A v1.0** — Linux Foundation standard (March 2026).
- **Symphony** — OpenAI production coding agent.
- **Agent Governance Toolkit v3.0.1** — Microsoft. Sub-millisecond policy enforcement.
- **Constitutional Evolution** — Evolved norms for multi-agent coordination.
- **REFLECT** — Transparent principle-guided reasoning.
- **InternAgent-1.5** — Shanghai AI Lab. Autonomous scientific discovery.
- **Temporal + OpenAI integration** — Durable execution GA.
- **DBOS MCP server** — Database-native agent execution.
