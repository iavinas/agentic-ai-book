# Appendix B: Benchmarks and Leaderboards

## Comprehensive Benchmark Catalog

### Coding
- **SWE-bench Verified** — 500 tasks. https://www.swebench.com/
- **SWE-bench Pro** — 731 harder tasks.
- **SWE-bench Live** — Monthly fresh issues.
- **HumanEval** — Synthetic functions. (Deprecated for real-world evaluation)

### Web & OS
- **WebArena Verified** — Web navigation. https://webarena.dev/
- **OSWorld-Verified** — Computer use. https://os-world.github.io/
- **Mind2Web** — Diverse web tasks. https://mind2web.github.io/

### General Reasoning
- **GAIA / GAIA2** — Multi-step reasoning. https://arxiv.org/abs/2311.12983
- **AgentBench** — 8 environments. https://github.com/THUDM/AgentBench
- **MINT** — Multi-turn interaction. https://arxiv.org/abs/2303.XXXXX

### Customer Service
- **Tau2-Bench** — Policy compliance.
- **CLUE** — Conversational understanding.

### Function Calling
- **BFCL V4** — Berkeley Function Calling Leaderboard.
- **MMAU, Agent-X, VAKRA, REXBench** — Function calling variants.

### Evaluation Frameworks
- **CLEAR Framework** — Accuracy, Cost, Latency, Reliability, Safety.
- **ReliabilityBench** — Consistency, perturbation tolerance, fault tolerance.
